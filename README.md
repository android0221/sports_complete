# Sports Complete

项目为体育相关的直播和数据App，数据来源是 www.zhibo8.cc 。

**项目持续维护中，欢迎Star、Fork，提Issue。**

## 开发环境

1. Flutter 2.5.3 • channel stable
2. Dart 2.14.4

## 已实现功能

* FIFA，男女足世界排名。
* 深色模式切换。
* 所有联赛和项目的赛程。
